# FISHLOCK PASSWORD CHECKER

A simple password checking application developed in python using GooeyPie for a lightweight GUI.

## Installation

Install my-project with npm

GooeyPie:

    pip install gooeypie
    pip3 install gooeypie

PYHIBP:

    pip install pyhibp
    pip3 install pyhibp
